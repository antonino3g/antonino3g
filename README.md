## Code snippet about me 
```dart 
class Me {
  final String name;
  String course;
  String university;
  String job;
  String focus;
  final String pronouns;

  Me({
    required this.name,
    required this.course,
    required this.university,
    required this.job,
    required this.focus,
    required this.pronouns,
  });

  String whoIAm() {
    return '''
    👨🏻‍💻 My name is $name
    🎓 I graduated in $course from $university
    📚 My time is being spent learning $focus
    😎 Pronouns: $pronouns
    ''';
  }
}

void main() {
  var antonino = Me(
    name: "Antonino Praxedes",
    course: "Systems Analysis and Development",
    university: "Federal Institute of Roraima",
    job: "Mobile Developer",
    focus: "Flutter development and Java Spring Boot",
    pronouns: "He/Him",
  );

  print(antonino.whoIAm());
}
```

    😎 Pronouns: He/Him
